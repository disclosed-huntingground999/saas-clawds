# Remote Work

## Overview

This folder is for **remote work policy** — remote-first vs hybrid, async communication, tools, and team rituals. Clear remote policies reduce ambiguity and support productivity.

## Questions to Answer

1. **What's your work model?** (Remote-first? Hybrid? Office-first?)
2. **Where can people work?** (Anywhere? Same timezone? Approved countries?)
3. **What are core hours?** (Overlap for meetings? Flexible?)
4. **What tools do you use?** (Slack, Zoom, Notion?)
5. **What's your async communication norm?** (Default async? Document first?)
6. **What rituals do you have?** (All-hands, team sync, 1-on-1s?)
7. **Do you have an office?** (Optional? Required for some?)

## Output Template

Create a `README.md` in this folder with:

```markdown
# Remote Work: [Your SaaS Name]

## Work Model
- **Model:** [Remote-first / Hybrid / Office-first]
- **Definition:** 
- **Office (if any):** [Location, optional/required]

## Location Policy
| Scenario | Policy |
|---|---|
| Hiring | [Anywhere / Same timezone / Approved countries] |
| Relocation | [Allowed / Case-by-case / No] |
| Travel | [Quarterly team offsites / Annual / None] |

## Core Hours
- **Overlap:** [e.g., 10am-2pm ET for meetings]
- **Flexibility:** [Outside overlap, flexible]
- **Time zones:** [e.g., US + EU covered]

## Tools
| Purpose | Tool |
|---|---|
| Chat | Slack |
| Video | Zoom / Meet |
| Docs | Notion / Google |
| Project management | Linear / Jira |
| Async video | Loom |

## Async Norms
- **Default:** Async first; meeting when needed
- **Response time:** [e.g., Same day for urgent]
- **Documentation:** [Decisions in Notion, not Slack]
- **Meeting-free blocks:** [e.g., No-meeting Wednesdays]

## Rituals
| Ritual | Cadence | Purpose |
|---|---|---|
| All-hands | Weekly / Bi-weekly | Company update |
| Team sync | Weekly | Team alignment |
| 1-on-1s | Weekly | Manager-employee |
| Retro | Sprint / Monthly | Improvement |
| Offsite | Quarterly / Annual | Bonding, strategy |
```
