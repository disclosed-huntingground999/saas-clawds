# Performance Management

## Overview

This folder is for **performance reviews, feedback, and growth plans** — how you evaluate performance, give feedback, and support development. Good performance management drives growth and retention.

## Questions to Answer

1. **What's your review cadence?** (Annual? Bi-annual? Quarterly?)
2. **What's your review format?** (Self-review? Manager review? 360?)
3. **How do you give continuous feedback?** (1-on-1s? Real-time?)
4. **Do you use performance ratings?** (1-5? Exceeds/Meets/Needs improvement?)
5. **How do you handle underperformance?** (PIP? Coaching?)
6. **Do you have growth plans or career paths?**
7. **What tools do you use?** (Lattice, 15Five, spreadsheet?)

## Output Template

Create a `README.md` in this folder with:

```markdown
# Performance Management: [Your SaaS Name]

## Review Cycle
| Cycle | Timing | Components |
|---|---|---|
| Annual | [Month] | Self, manager, calibration |
| Mid-year | [Month] | Check-in, goals |
| Quarterly | Ongoing | 1-on-1s, progress |
| Continuous | Weekly | 1-on-1s, ad-hoc feedback |

## Review Process
| Step | Owner | Duration |
|---|---|---|
| Self-review | Employee | 1 week |
| Manager review | Manager | 1 week |
| Calibration | Leadership | 1 day |
| Feedback meeting | Manager | 1:1 |
| Growth plan | Employee + Manager | Follow-up |

## Feedback Framework
- **When:** Real-time, weekly 1-on-1s
- **Format:** Situation-Behavior-Impact (SBI) or similar
- **Culture:** Radical Candor (Care personally, challenge directly)
- **Values alignment:** [Link to culture-values]

## Ratings (If Used)
| Rating | Definition |
|---|---|
| Exceeds | Consistently surpasses expectations |
| Meets | Meets expectations |
| Needs Improvement | Below expectations, plan required |

## Underperformance
| Step | Action |
|---|---|
| 1 | Conversation, clarify expectations |
| 2 | Written improvement plan (PIP) |
| 3 | 30-60-90 day check-ins |
| 4 | Outcome: improve or exit |
```
